NodeJs의 주요 확장 모듈 - socket.io

socket.io를 이용한 채팅 구현

socket.io를 사용해서 실시간 채팅 프로그램을 만들어본다.

주요기능

1. 접속하면 알림 출력
2. 닉네임 변경 가능
3. 닉네임 변경했을 때 알림 출력
4. 유저가 떠났을 경우 알림 출력

=================================================

온라인 빙고 게임 만들기

 5 X 5 칸을 그린 후 번갈아 가면서 숫자를 부르고, 해당 숫자들을 지워 가로나 세로, 대각선으로 먼저 빙고를 만드는 사람이 이기는 빙고 게임을 만들어본다.
 
1. 현재 접속 중인 유저 이름 출력
2. 한 명이 게임을 시작하려 하면 불가능하게 막기, 알림 출력
3. 게임 진행을 위해 턴 넘기기
4. 자기 턴이 아닐 경우 진행 불가능, 알림 출력
5. 숫자가 선택되면 상대방에게 알림 출력
6. 선택된 숫자에 글자 효과 주기
